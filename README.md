## What's included:
- react
- Alt.js
- Karma
- Babel
- webpack

## How you use it

You need `webpack` and `webpack-dev-server` installed globally.

```
$ npm install webpack -g
```
```
$ npm install webpack-dev-server -g
```

To install deps:
```
$ npm install
```

To run tests:
```
$ karma start
```

To see the result:
```
$ webpack
```
then open `index.html`

Or, easier
```
$ webpack-dev-server
```
then open `http://localhost:8080/webpack-dev-server/`
